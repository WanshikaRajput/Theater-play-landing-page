# Theater Play - Static Landing Page

A modern, responsive static landing page developed using pure HTML5, CSS3, and Vanilla JavaScript. This project was created as part of an internship assignment to demonstrate responsive UI design, accessibility, form validation, and clean front-end development practices.
## Live url


## Features

- Responsive design for desktop, tablet, and mobile devices
- Fixed navigation bar with smooth scrolling
- Modern hero section with illustration
- About section with feature highlights
- Service cards with hover animations
- Contact form with client-side validation
- Email format validation
- XSS input sanitization
- Loading simulation with spinner
- Empty state handling
- Dark mode toggle
- Accessibility support using semantic HTML and ARIA labels
- Simulated analytics logging

---

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

No external frameworks or libraries were used.

## Functional Requirements Implemented

### User Experience

- Responsive landing page
- Fast user interaction
- Smooth navigation

### Form Validation

- Required field validation
- Email validation
- Invalid input highlighting

### Loading State

- Simulated asynchronous loading
- Visual loading spinner

### Empty State

- Displays a user-friendly "No data found" message

### Security

- Basic XSS sanitization before storing user input

### Accessibility

- Semantic HTML
- ARIA labels
- Keyboard navigable interface

### Telemetry Simulation

Logs the following message after primary interactions:

```
[Analytics] User interacted with Static Landing Page
```
